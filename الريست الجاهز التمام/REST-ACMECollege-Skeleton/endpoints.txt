1. STUDENT ENDPOINTS (/student)
================================================================================

1.1 GET All Students
   Method: GET
   URL: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/student
   Auth: ADMIN_ROLE
   Description: Get all students

1.2 GET Student by ID
   Method: GET
   URL: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/student/{id}
   Example: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/student/1
   Auth: ADMIN_ROLE, USER_ROLE
   Description: Get specific student by ID

1.3 POST Create Student
   Method: POST
   URL: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/student
   Auth: ADMIN_ROLE
   Content-Type: application/json
   Body Example:
   {
     "firstName": "John",
     "lastName": "Doe",
     "email": "john.doe@example.com",
     "phoneNumber": "123-456-7890",
     "program": {
       "id": 1
     }
   }

1.4 PUT Update Student
   Method: PUT
   URL: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/student/{id}
   Example: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/student/1
   Auth: ADMIN_ROLE
   Content-Type: application/json
   Body Example:
   {
     "firstName": "John",
     "lastName": "Doe Updated",
     "email": "john.doe@example.com",
     "phoneNumber": "123-456-7890"
   }

1.5 DELETE Student
   Method: DELETE
   URL: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/student/{id}
   Example: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/student/1
   Auth: ADMIN_ROLE
   Description: Delete student by ID

1.6 GET All Programs
   Method: GET
   URL: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/student/program
   Auth: ADMIN_ROLE, USER_ROLE
   Description: Get all programs

5. COURSE REGISTRATION ENDPOINTS (/courseregistration)
================================================================================

5.1 GET All Course Registrations
   Method: GET
   URL: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/courseregistration
   Auth: ADMIN_ROLE
   Description: Get all course registrations

5.2 GET Course Registration by Student ID and Course ID
   Method: GET
   URL: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/courseregistration/student/{studentId}/course/{courseId}
   Example: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/courseregistration/student/1/course/2
   Auth: ADMIN_ROLE, USER_ROLE
   Description: Get specific course registration

5.3 POST Create Course Registration
   Method: POST
   URL: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/courseregistration
   Auth: ADMIN_ROLE
   Content-Type: application/json
   Body Example:
   {
     "student": {
       "id": 1
     },
     "course": {
       "id": 2
     },
     "year": 2024,
     "semester": "FALL"
   }

5.4 PUT Assign Professor to Course Registration
   Method: PUT
   URL: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/courseregistration/student/{studentId}/course/{courseId}
   Example: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/courseregistration/student/1/course/2
   Auth: ADMIN_ROLE
   Content-Type: application/json
   Body Example:
   {
     "id": 1
   }
   Note: Send only the professor ID in the body

5.5 PUT Assign Grade to Course Registration
   Method: PUT
   URL: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/courseregistration/student/{studentId}/course/{courseId}/lettergrade
   Example: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/courseregistration/student/1/course/2/lettergrade
   Auth: ADMIN_ROLE
   Content-Type: application/json
   Body Example:
   "A+"
   Note: Send only the letter grade string (e.g., "A+", "B", "C-")

5.6 DELETE Course Registration
   Method: DELETE
   URL: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/courseregistration/student/{studentId}/course/{courseId}
   Example: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/courseregistration/student/1/course/2
   Auth: ADMIN_ROLE
   Description: Delete course registration

5.7 GET All Letter Grades
   Method: GET
   URL: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1/courseregistration/lettergrade
   Auth: ADMIN_ROLE
   Description: Get all available letter grades

POSTMAN SETUP INSTRUCTIONS
================================================================================

1. Create a new Collection in Postman named "ACME College API"

2. Set Collection Variables:
   - base_url: http://localhost:8080/REST-ACMECollege-Skeleton/api/v1
   - admin_username: admin
   - admin_password: admin
   - user_username: cst8277
   - user_password: 8277

3. For each request:
   - Go to Authorization tab
   - Select Type: Basic Auth
   - Username: {{admin_username}} or {{user_username}}
   - Password: {{admin_password}} or {{user_password}}

4. For POST/PUT requests:
   - Go to Headers tab
   - Add: Content-Type: application/json
   - Go to Body tab
   - Select: raw
   - Select: JSON
   - Paste the JSON body example

5. Replace {id}, {studentId}, {courseId} in URLs with actual values
